Bab 3: Sistem Komputer

Sistem komputer adalah gabungan perangkat keras, perangkat lunak, dan pengguna yang bekerja untuk mengolah data.

1. Komponen Sistem Komputer

  • Hardware (Perangkat Keras): CPU, RAM, Hard Drive, Mouse, Keyboard.
  • Software (Perangkat Lunak): Sistem operasi (Windows, Linux), Aplikasi (Ms. Office, Photoshop).
  • Brainware: Pengguna yang mengoperasikan komputer.

2. Proses Kerja Komputer

  • Input → Proses → Output
  • Data dimasukkan melalui input (keyboard, mouse), diproses oleh CPU, lalu ditampilkan pada output (monitor, printer).
